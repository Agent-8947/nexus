## ⚙️ 2. BACKEND — API и бизнес-логика

*Цель: Надежные API, интеграции и серверные вычисления.*

| Категория | Технология | Роль |
| :--- | :--- | :--- |
| **Основной** | **FastAPI (Python)** | Масштабируемые API. AI знает досконально. Авто-дока. |
| **Edge API** | **Hono (TS)** | Сверхлегкий (14KB). Работает на Cloudflare Workers. |

| **Деплой** | **Railway / Render** | "Vercel для бэкенда". Контейнеризация и деплой. |

## 🤖 3. AI / АВТОМАТИЗАЦИЯ — Агенты и пайплайны

*Цель: Создание автономных инструментов и интеграция LLM.*

| Категория | Технология | Роль |
| :--- | :--- | :--- |
