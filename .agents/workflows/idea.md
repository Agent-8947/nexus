Этот ворклоу активирует протокол "Nexus Idea Capture" для мгновенного сохранения гипотез, архитектурных набросков и творческих мыслей.

### 1. Как использовать

Введите `/idea <ваша идея>` для быстрой записи.

### 2. Структура хранения

- Все идеи сохраняются в файл `PROJECT/IDEA/LOG.md`.
- Каждая запись получает метку времени и уникальный ID (NIX-ID).
- Идеи автоматически классифицируются по категориям (UI/UX, Backend, Strategy, Luxury Art).

### 3. Автоматизация (Execution)

// turbo
python PROJECT/IDEA/scripts/process_idea.py "{{args}}"

### 4. Анализ (Review)

Раз в неделю агент анализирует `LOG.md` и предлагает архитектурные паттерны для реализации самых перспективных идей.
